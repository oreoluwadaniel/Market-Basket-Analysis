# Market Basket Analysis

Association rule mining on retail transaction data to uncover which products customers buy together and how demand shifts across the year.

## Overview

This project analyzes transactions from an online retail store to surface patterns useful for cross-selling, bundling, and inventory planning. The dataset includes invoice numbers, stock codes, product descriptions, quantities, prices, customer IDs, and countries.

## Objectives

- Identify product pairs frequently bought together
- Analyze seasonal trends for top-selling products
- Turn both into recommendations for inventory, bundling, and marketing

## Tech stack

Python, pandas, mlxtend (association rules)

## Key findings

- Products like "WHITE HANGING HEART T-LIGHT HOLDER" and "REGENCY CAKESTAND 3 TIER" show strong purchase associations, pointing to clear cross-sell and bundling opportunities.
- Several top-selling products show sharp seasonal peaks, useful for planning promotions and inventory ahead of demand.
